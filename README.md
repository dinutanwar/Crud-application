# 📌 CRUD Web Application

## 📖 Description

This project is a simple and interactive **CRUD (Create, Read, Update, Delete) Web Application** built using HTML, CSS, and JavaScript. It allows users to manage data efficiently through a clean user interface.

## 🚀 Features

* ➕ Add new user details
* 👀 View user information
* ✏️ Edit existing records
* ❌ Delete user data
* 🔄 Fetch data from API
* 📊 Dynamic table display

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript (ES6)
* Fetch API

## 🌐 API

* JSONPlaceholder (Fake REST API)

## ▶️ How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
